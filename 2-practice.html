Задание 3. Простой калькулятор

Реализовать простой калькулятор, пользуясь делегированием. (Делегирование в javascript – это возможность обрабатывать дочерние элементы через один родительский) Все кнопки должны быть функциональны и калькулятор должен вычислять по нажатию на знак равно.

Калькулятор работает только с двумя операндами
Учесть:

  • По нажатию на любую из кнопок мы видим её на экране (внутри тега input)
  • Кнопка "x²" возводит во вторую степень введённое число
  • Кнопка "±" меняет знак операнда на противоположный
  • Кнопка "Сброс" возвращает калькулятор в начальное состояние
  • * Если нажали на 7, потом на знак, например, +, а затем вместо цифры нажали на любой другой знак, например "-", то вместо сложение будет происходить вычитание, т.к. знак действия изменился
